## Useful commands

Start services:
```console 
foo@bar:~$ docker-compose up -d
```

Connect to postgres service console:
```console            
foo@bar:~$ docker-compose exec postgres sh
#
``` 

Connect to PostgreSQL:
```console
# psql -U dev
psql (14.6 (Debian 14.6-1.pgdg110+1))
Type "help" for help.

dev=#
```
